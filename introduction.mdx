## Real-Time Chat + AI + DevOps-Ready

OpsChat is a modern real-time chat platform that combines instant messaging with powerful AI capabilities, designed with DevOps teams in mind. Built with React, Node.js, Socket.io, and PostgreSQL, it's production-ready and scales horizontally with Kubernetes.

## Why OpsChat?

<AccordionGroup>
  <Accordion title="Real-Time Performance" icon="bolt">
    Built on Socket.io with Redis pub/sub for instant message delivery across multiple server instances. Messages are delivered in milliseconds with automatic reconnection and presence tracking.
  </Accordion>

  <Accordion title="AI-Powered Communication" icon="brain">
    Integrated Groq LLM provides:
    - **Smart Translation**: Translate messages into 50+ languages
    - **Conversation Summarization**: Generate concise summaries of long discussions
    - **Smart Reply Suggestions**: AI-generated contextual responses
  </Accordion>

  <Accordion title="Production-Grade Stack" icon="layer-group">
    - **Frontend**: React 19, TypeScript, Tailwind CSS, Vite
    - **Backend**: Node.js, Express, Socket.io, Prisma ORM
    - **Database**: PostgreSQL with connection pooling
    - **Cache**: Redis for session management and pub/sub
    - **Storage**: MinIO (S3-compatible) for file uploads
    - **Orchestration**: Kubernetes-ready with horizontal scaling
  </Accordion>

  <Accordion title="DevOps-First Design" icon="truck">
    - Docker Compose for local development
    - Kubernetes manifests included
    - Horizontal scaling with Redis adapter
    - Health checks and graceful shutdown
    - CI/CD ready with GitHub Actions
  </Accordion>
</AccordionGroup>

## Tech Stack at a Glance

| Layer | Technology |
|-------|------------|
| Frontend | React 19, TypeScript, Vite, Tailwind CSS |
| Backend | Node.js, Express, Socket.io, Prisma |
| Database | PostgreSQL 15 |
| Cache | Redis |
| Storage | MinIO (S3-compatible) |
| AI | Groq API (LLaMA 3.3) |
| Container | Docker, Docker Compose |
| Orchestration | Kubernetes (KIND for local) |
